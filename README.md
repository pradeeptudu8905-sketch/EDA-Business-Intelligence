📊 Exploratory Data Analysis (EDA) & Business Intelligence – Netflix Dataset
🔹 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) and answering business-driven questions using SQL on the Netflix dataset.
The goal is to uncover patterns, trends, and insights that can support better business decision-making.

🎯 Objective
Analyze Netflix data to understand content distribution and trends
Perform data cleaning and preprocessing
Apply statistical analysis and visualization techniques
Solve real-world business questions using SQL
Create a static dashboard for key performance indicators (KPIs)
🛠️ Tools & Technologies Used
Python (Pandas, Matplotlib, Seaborn)
SQL (MySQL)
Microsoft Excel (Dashboard Creation)
Jupyter Notebook
📂 Dataset
Netflix Movies and TV Shows Dataset
Includes information such as title, country, director, release year, rating, duration, and date added
